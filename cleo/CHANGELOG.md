# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-23

### Added
- Initial release of Cleo Household Manager
- Telegram bot integration with command support
- Google Gemini AI integration for natural language understanding
- Multi-tier LLM routing (Flash for simple, Pro for complex queries)
- Chore tracking with automatic due date calculation
- Chore management commands: /list, /done, /add, /remove
- Automatic reminder system with configurable intervals
- SQLite database for persistent storage
- Chat history tracking for conversation context
- Security middleware to restrict bot access to specific chat IDs
- Comprehensive logging with configurable log levels
- Default chores: vacuuming, litterbox, trash, dishes
- Complete documentation (README.md, DOCS.md)
- Home Assistant add-on packaging (config.yaml, Dockerfile, run.sh)

### Features
- Natural conversation with AI assistant
- Action parsing for automatic chore completion
- Graceful shutdown handling
- Database migrations
- Rate limiting for reminders
- Emoji indicators for chore status
- Markdown formatting in bot responses
